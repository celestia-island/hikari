
## واجهة البرمجة

| الخاصية | الوصف | النوع | الافتراضي |
|---------|-------|-------|-----------|
| author | اسم المؤلف | String | - |
| avatar | رابط الصورة الرمزية | Option\<String\> | None |
| content | محتوى التعليق | Element | - |
| datetime | التاريخ والوقت | Option\<String\> | None |
