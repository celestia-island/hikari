# التدويل i18n

نظام التدويل في هيكاري يدعم تبديل لغات متعددة مع دعم كامل لـ RTL (من اليمين إلى اليسار).

## اللغات المدعومة

| اللغة | الرمز | الاتجاه |
|-------|-------|---------|
| الإنجليزية | `en-US` | من اليسار |
| الصينية المبسطة | `zh-CHS` | من اليسار |
| الصينية التقليدية | `zh-CHT` | من اليسار |
| الفرنسية | `fr-FR` | من اليسار |
| الروسية | `ru-RU` | من اليسار |
| الإسبانية | `es-ES` | من اليسار |
| العربية | `ar-SA` | **من اليمين** |
| اليابانية | `ja-JP` | من اليسار |
| الكورية | `ko-KR` | من اليسار |

## الاستخدام الأساسي

## دعم RTL

بالنسبة للغات RTL مثل العربية، يقوم `I18nProvider` بتعيين `dir="rtl"` تلقائياً:

```rust
use tairitsu_web::i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // RTL تلقائياً
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

تتكيف مكونات التخطيط تلقائياً مع اتجاه RTL.

## API

| الخاصية | الوصف | النوع | الافتراضي |
|---------|-------|-------|-----------|
| language | اللغة الحالية | Language | الإنجليزية |
| toml_content | محتوى الترجمة | &str | - |
| direction | اتجاه التخطيط (ThemeProvider) | "ltr" \| "rtl" | "ltr" |
