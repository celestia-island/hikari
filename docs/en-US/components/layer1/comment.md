# Comment

Comment component for displaying comment content.

## Basic Usage

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| author | Author name | String | - |
| avatar | Avatar URL | Option\<String\> | None |
| content | Comment content | Element | - |
| datetime | Datetime | Option\<String\> | None |
