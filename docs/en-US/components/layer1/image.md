# Image

Image component for displaying images with loading state and error handling.

## Basic Usage

```_hikari_component
pages/components/layer1/image#basic
```

## Loading Placeholder

```_hikari_component
pages/components/layer1/image#placeholder
```

## API

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| src | Image URL | String | - |
| alt | Alt text | String | - |
| width | Width | Option\<String\> | None |
| height | Height | Option\<String\> | None |
