# Pagination

Pagination component for data paging.

## Basic Usage

```_hikari_component
pages/components/layer2/pagination#basic
```

## API

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| total | Total count | usize | 0 |
| page_size | Items per page | usize | 10 |
| current | Current page | usize | 1 |
