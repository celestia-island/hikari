# i18n Internationalization

Hikari's internationalization system supports multiple language switching.

## Basic Usage

```_hikari_component
pages/system/i18n#basic
```

## Language Switching

```_hikari_component
pages/system/i18n#switch
```

## API

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| language | Current language | Language | English |
| toml_content | Translation content | &str | - |
