# ZoomControls

Zoom control component for interface scaling.

## Basic Usage

```_hikari_component
pages/components/layer3/zoom_controls#basic
```

## API

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| zoom | Zoom level | f64 | 1.0 |
| min | Minimum | f64 | 0.5 |
| max | Maximum | f64 | 2.0 |
