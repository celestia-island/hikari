# Comment

Componente Comment para mostrar contenido de comentarios.

## Uso Básico

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| Propiedad | Descripción | Tipo | Por Defecto |
|-----------|-------------|------|-------------|
| author | Nombre del autor | String | - |
| avatar | URL del avatar | Option\<String\> | None |
| content | Contenido del comentario | Element | - |
| datetime | Fecha y hora | Option\<String\> | None |
