# ZoomControls

Componente de control de zoom para escalado de interfaz.

## Uso Básico

```_hikari_component
pages/components/layer3/zoom_controls#basic
```

## API

| Propiedad | Descripción | Tipo | Por Defecto |
|-----------|-------------|------|-------------|
| zoom | Nivel de zoom | f64 | 1.0 |
| min | Mínimo | f64 | 0.5 |
| max | Máximo | f64 | 2.0 |
