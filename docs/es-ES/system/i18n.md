# i18n Internacionalización

El sistema de internacionalización de Hikari soporta múltiples idiomas con soporte completo RTL (Derecha a Izquierda).

## Idiomas Soportados

| Idioma | Código | Dirección |
|--------|--------|-----------|
| Inglés | `en-US` | LTR |
| Chino Simplificado | `zh-CHS` | LTR |
| Chino Tradicional | `zh-CHT` | LTR |
| Francés | `fr-FR` | LTR |
| Ruso | `ru-RU` | LTR |
| Español | `es-ES` | LTR |
| Árabe | `ar-SA` | **RTL** |
| Japonés | `ja-JP` | LTR |
| Coreano | `ko-KR` | LTR |

## Uso Básico

```_hikari_component
pages/system/i18n#basic
```

## Cambio de Idioma

```_hikari_component
pages/system/i18n#switch
```

## Soporte RTL

Para idiomas RTL como el árabe, `I18nProvider` establece automáticamente `dir="rtl"`:

```rust
use hikari_i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // Automáticamente RTL
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

Los componentes de diseño se adaptan automáticamente a la dirección RTL.

## API

| Propiedad | Descripción | Tipo | Por Defecto |
|-----------|-------------|------|-------------|
| language | Idioma actual | Language | English |
| toml_content | Contenido de traducción | &str | - |
| direction | Dirección del diseño (ThemeProvider) | "ltr" \| "rtl" | "ltr" |
