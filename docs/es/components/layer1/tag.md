# Tag

Componente Tag para mostrar etiquetas o marcas.

## Uso Básico

```_hikari_component
pages/components/layer1/tag#basic
```

## Etiqueta Cerrable

```_hikari_component
pages/components/layer1/tag#closable
```

## API

| Propiedad | Descripción | Tipo | Por Defecto |
|-----------|-------------|------|-------------|
| children | Contenido de la etiqueta | Element | - |
| closable | Si es cerrable | bool | false |
| on_close | Callback de cierre | Option\<EventHandler\> | None |
