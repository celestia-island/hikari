# Commentaire

Composant de commentaire pour afficher le contenu des commentaires.

## Utilisation de base

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| Propriété | Description | Type | Défaut |
|----------|-------------|------|--------|
| author | Nom de l'auteur | String | - |
| avatar | URL de l'avatar | Option\<String\> | Aucun |
| content | Contenu du commentaire | Element | - |
| datetime | Date et heure | Option\<String\> | Aucun |
