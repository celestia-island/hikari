# Tag

Composant Tag pour afficher des étiquettes ou des marques.

## Utilisation de base

```_hikari_component
pages/components/layer1/tag#basic
```

## Tag fermable

```_hikari_component
pages/components/layer1/tag#closable
```

## API

| Propriété | Description | Type | Défaut |
|----------|-------------|------|--------|
| children | Contenu du tag | Element | - |
| closable | Fermable | bool | false |
| on_close | Rappel de fermeture | Option\<EventHandler\> | Aucun |
