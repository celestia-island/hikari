# Contrôles de zoom

Composant de contrôle de zoom pour la mise à l'échelle de l'interface.

## Utilisation de base

```_hikari_component
pages/components/layer3/zoom_controls#basic
```

## API

| Propriété | Description | Type | Défaut |
|----------|-------------|------|--------|
| zoom | Niveau de zoom | f64 | 1.0 |
| min | Minimum | f64 | 0.5 |
| max | Maximum | f64 | 2.0 |
