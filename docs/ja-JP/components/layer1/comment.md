# コメント

コメントコンポーネントは、コメント内容を表示するために使用されます。

## 基本的な使い方

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| プロパティ | 説明 | 型 | デフォルト値 |
|----------|-------------|------|---------|
| author | 作成者名 | String | - |
| avatar | アバターURL | Option\<String\> | None |
| content | コメント内容 | Element | - |
| datetime | 日時 | Option\<String\> | None |
