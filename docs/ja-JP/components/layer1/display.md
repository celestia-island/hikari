# 表示コンポーネント

基本的なデータ表示コンポーネント。

## バッジ

ステータスやカウントを表示するバッジコンポーネント。

```_hikari_component
pages/components/layer1/display#badge
```

## 区切り線

コンテンツを区切るための区切り線コンポーネント。

```_hikari_component
pages/components/layer1/display#divider
```

## カード

コンテンツをグループ化して表示するカードコンポーネント。

```_hikari_component
pages/components/layer1/display#card
```
