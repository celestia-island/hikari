# 画像

画像コンポーネントは、読み込み状態とエラー処理付きで画像を表示します。

## 基本的な使い方

```_hikari_component
pages/components/layer1/image#basic
```

## 読み込みプレースホルダー

```_hikari_component
pages/components/layer1/image#placeholder
```

## API

| プロパティ | 説明 | 型 | デフォルト値 |
|----------|-------------|------|---------|
| src | 画像URL | String | - |
| alt | 代替テキスト | String | - |
| width | 幅 | Option\<String\> | None |
| height | 高さ | Option\<String\> | None |
