# CSSユーティリティ

Tailwind互換のユーティリティクラスで、迅速なスタイリングを実現します。

## 表示

要素の表示動作を制御します。

```_hikari_component
pages/system/css#display
```

## フレックスボックス

フレックスレイアウトユーティリティクラス。

```_hikari_component
pages/system/css#flex
```

## スペーシング

スペーシングユーティリティクラス（パディング、マージン、ギャップ）。

```_hikari_component
pages/system/css#spacing
```

## タイポグラフィ

タイポグラフィユーティリティクラス。

```_hikari_component
pages/system/css#typography
```

## カラー

カラーユーティリティクラス。

```_hikari_component
pages/system/css#colors
```
