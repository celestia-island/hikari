# i18n 国際化

Hikariの国際化システムは、完全なRTL（右から左へ）サポートを備えた多言語切り替えをサポートしています。

## サポート言語

| 言語 | コード | 方向 |
|------|--------|------|
| 英語 | `en-US` | LTR |
| 簡体字中国語 | `zh-CHS` | LTR |
| 繁体字中国語 | `zh-CHT` | LTR |
| フランス語 | `fr-FR` | LTR |
| ロシア語 | `ru-RU` | LTR |
| スペイン語 | `es-ES` | LTR |
| アラビア語 | `ar-SA` | **RTL** |
| 日本語 | `ja-JP` | LTR |
| 韓国語 | `ko-KR` | LTR |

## 基本的な使用方法

```_hikari_component
pages/system/i18n#basic
```

## 言語切り替え

```_hikari_component
pages/system/i18n#switch
```

## RTL サポート

アラビア語などのRTL言語の場合、`I18nProvider`は自動的に`dir="rtl"`を設定します:

```rust
use hikari_i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // 自動的にRTL
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

レイアウトコンポーネントは自動的にRTL方向に適応します。

## API

| プロパティ | 説明 | 型 | デフォルト |
|------------|------|-----|------------|
| language | 現在の言語 | Language | 英語 |
| toml_content | 翻訳コンテンツ | &str | - |
| direction | レイアウト方向（ThemeProvider） | "ltr" \| "rtl" | "ltr" |
