# Display 컴포넌트

기본 데이터 표시 컴포넌트들입니다.

## Badge

상태 또는 개수를 표시하는 배지 컴포넌트입니다.

```_hikari_component
pages/components/layer1/display#badge
```

## Divider

콘텐츠를 구분하는 구분선 컴포넌트입니다.

```_hikari_component
pages/components/layer1/display#divider
```

## Card

콘텐츠를 그룹화하고 표시하는 카드 컴포넌트입니다.

```_hikari_component
pages/components/layer1/display#card
```
