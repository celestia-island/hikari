# CSS 유틸리티

Tailwind 호환 유틸리티 클래스로 빠른 스타일링을 지원합니다.

## Display

요소의 표시 동작을 제어합니다.

```_hikari_component
pages/system/css#display
```

## Flexbox

Flex 레이아웃 유틸리티 클래스들입니다.

```_hikari_component
pages/system/css#flex
```

## Spacing

간격 유틸리티 클래스들 (패딩, 마진, 간격)입니다.

```_hikari_component
pages/system/css#spacing
```

## Typography

타이포그래피 유틸리티 클래스들입니다.

```_hikari_component
pages/system/css#typography
```

## Colors

색상 유틸리티 클래스들입니다.

```_hikari_component
pages/system/css#colors
```
