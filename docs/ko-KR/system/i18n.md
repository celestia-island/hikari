# i18n 국제화

Hikari의 국제화 시스템은 완전한 RTL (오른쪽에서 왼쪽) 지원과 함께 다국어 전환을 지원합니다.

## 지원 언어

| 언어 | 코드 | 방향 |
|------|------|------|
| 영어 | `en-US` | LTR |
| 중국어 간체 | `zh-CHS` | LTR |
| 중국어 번체 | `zh-CHT` | LTR |
| 프랑스어 | `fr-FR` | LTR |
| 러시아어 | `ru-RU` | LTR |
| 스페인어 | `es-ES` | LTR |
| 아랍어 | `ar-SA` | **RTL** |
| 일본어 | `ja-JP` | LTR |
| 한국어 | `ko-KR` | LTR |

## 기본 사용법

```_hikari_component
pages/system/i18n#basic
```

## 언어 전환

```_hikari_component
pages/system/i18n#switch
```

## RTL 지원

아랍어와 같은 RTL 언어의 경우, `I18nProvider`가 자동으로 `dir="rtl"`을 설정합니다:

```rust
use hikari_i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // 자동으로 RTL
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

레이아웃 컴포넌트는 RTL 방향에 자동으로 적응합니다.

## API

| 속성 | 설명 | 타입 | 기본값 |
|------|------|------|--------|
| language | 현재 언어 | Language | English |
| toml_content | 번역 콘텐츠 | &str | - |
| direction | 레이아웃 방향 (ThemeProvider) | "ltr" \| "rtl" | "ltr" |
