# CSS 유틸리티

Tailwind 호환 유틸리티 클래스로 빠른 스타일링을 지원합니다.

## Display

요소의 표시 동작을 제어합니다.

```hikari
rsx! {
    div { style: "padding:1rem;color:#999;", "Component preview: pages/system/css#display" }
}
```
