# Комментарий

Компонент Comment для отображения содержимого комментариев.

## Базовое использование

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| Свойство | Описание | Тип | По умолчанию |
|----------|----------|-----|--------------|
| author | Имя автора | String | - |
| avatar | URL аватара | Option\<String\> | None |
| content | Содержимое комментария | Element | - |
| datetime | Дата и время | Option\<String\> | None |
