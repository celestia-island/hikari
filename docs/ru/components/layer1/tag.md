# Тег

Компонент Tag для отображения меток или отметок.

## Базовое использование

```_hikari_component
pages/components/layer1/tag#basic
```

## Закрываемый тег

```_hikari_component
pages/components/layer1/tag#closable
```

## API

| Свойство | Описание | Тип | По умолчанию |
|----------|----------|-----|--------------|
| children | Содержимое тега | Element | - |
| closable | Закрываемый | bool | false |
| on_close | Обратный вызов закрытия | Option\<EventHandler\> | None |
