# i18n Интернационализация

Система интернационализации Hikari поддерживает переключение между несколькими языками с полной поддержкой RTL (справа налево).

## Поддерживаемые языки

| Язык | Код | Направление |
|------|-----|-------------|
| Английский | `en-US` | LTR |
| Китайский упрощённый | `zh-CHS` | LTR |
| Китайский традиционный | `zh-CHT` | LTR |
| Французский | `fr-FR` | LTR |
| Русский | `ru-RU` | LTR |
| Испанский | `es-ES` | LTR |
| Арабский | `ar-SA` | **RTL** |
| Японский | `ja-JP` | LTR |
| Корейский | `ko-KR` | LTR |

## Базовое использование

```hikari
rsx! {
    div { style: "padding:1rem;color:#999;", "Component preview: pages/system/i18n#basic" }
}
```

## Переключение языка

```hikari
rsx! {
    div { style: "padding:1rem;color:#999;", "Component preview: pages/system/i18n#switch" }
}
```

## Поддержка RTL

Для RTL-языков, таких как арабский, `I18nProvider` автоматически устанавливает `dir="rtl"`:

```rust
use tairitsu_web::i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // Автоматически RTL
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

Компоненты раскладки автоматически адаптируются к RTL-направлению.

## API

| Свойство | Описание | Тип | По умолчанию |
|----------|----------|-----|--------------|
| language | Текущий язык | Language | English |
| toml_content | Содержимое перевода | &str | - |
| direction | Направление раскладки (ThemeProvider) | "ltr" \| "rtl" | "ltr" |
