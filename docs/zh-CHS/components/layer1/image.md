# Image 图片

Image 组件用于展示图片，支持加载状态和错误处理。

## 基础用法

```_hikari_component
pages/components/layer1/image#basic
```

## 加载占位

```_hikari_component
pages/components/layer1/image#placeholder
```

## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| src | 图片地址 | String | - |
| alt | 替代文字 | String | - |
| width | 宽度 | Option\<String\> | None |
| height | 高度 | Option\<String\> | None |
