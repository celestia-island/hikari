# Tag 标签

Tag 组件用于展示标签或标记。

## 基础用法

```_hikari_component
pages/components/layer1/tag#basic
```

## 可关闭标签

```_hikari_component
pages/components/layer1/tag#closable
```

## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| children | 标签内容 | Element | - |
| closable | 可关闭 | bool | false |
| on_close | 关闭回调 | Option\<EventHandler\> | None |
