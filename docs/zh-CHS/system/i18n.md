# i18n 国际化

Hikari 的国际化系统支持多语言切换，并提供完整的 RTL（从右到左）支持。

## 支持的语言

| 语言 | 代码 | 方向 |
|------|------|------|
| 英语 | `en-US` | LTR |
| 简体中文 | `zh-CHS` | LTR |
| 繁体中文 | `zh-CHT` | LTR |
| 法语 | `fr-FR` | LTR |
| 俄语 | `ru-RU` | LTR |
| 西班牙语 | `es-ES` | LTR |
| 阿拉伯语 | `ar-SA` | **RTL** |
| 日语 | `ja-JP` | LTR |
| 韩语 | `ko-KR` | LTR |

## 基础用法

```_hikari_component
pages/system/i18n#basic
```

## 语言切换

```_hikari_component
pages/system/i18n#switch
```

## RTL 支持

对于阿拉伯语等 RTL 语言，`I18nProvider` 会自动设置 `dir="rtl"`：

```rust
use hikari_i18n::{I18nProvider, context::Language};

fn App() -> Element {
    rsx! {
        I18nProvider {
            language: Language::Arabic,  // 自动 RTL
            toml_content: AR_SA_TOML,
            YourApp {}
        }
    }
}
```

布局组件会自动适应 RTL 方向。

## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| language | 当前语言 | Language | English |
| toml_content | 翻译内容 | &str | - |
| direction | 布局方向（ThemeProvider） | "ltr" \| "rtl" | "ltr" |
