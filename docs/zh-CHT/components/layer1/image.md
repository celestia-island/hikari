# Image 圖片

Image 元件用於展示圖片，支援載入狀態和錯誤處理。

## 基礎用法

```_hikari_component
pages/components/layer1/image#basic
```

## 載入佔位

```_hikari_component
pages/components/layer1/image#placeholder
```

## API

| 屬性 | 說明 | 類型 | 預設值 |
|------|------|------|--------|
| src | 圖片位址 | String | - |
| alt | 替代文字 | String | - |
| width | 寬度 | Option\<String\> | None |
| height | 高度 | Option\<String\> | None |
