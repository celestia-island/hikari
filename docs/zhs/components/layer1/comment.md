# Comment 评论

Comment 组件用于展示评论内容。

## 基础用法

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| author | 作者名 | String | - |
| avatar | 头像地址 | Option\<String\> | None |
| content | 评论内容 | Element | - |
| datetime | 时间 | Option\<String\> | None |
