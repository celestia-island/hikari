# Comment 評論

Comment 元件用於展示評論內容。

## 基礎用法

```_hikari_component
pages/components/layer1/comment#basic
```

## API

| 屬性 | 說明 | 類型 | 預設值 |
|------|------|------|--------|
| author | 作者名 | String | - |
| avatar | 頭像位址 | Option\<String\> | None |
| content | 評論內容 | Element | - |
| datetime | 時間 | Option\<String\> | None |
