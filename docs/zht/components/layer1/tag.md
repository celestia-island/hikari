# Tag 標籤

Tag 元件用於展示標籤或標記。

## 基礎用法

```_hikari_component
pages/components/layer1/tag#basic
```

## 可關閉標籤

```_hikari_component
pages/components/layer1/tag#closable
```

## API

| 屬性 | 說明 | 類型 | 預設值 |
|------|------|------|--------|
| children | 標籤內容 | Element | - |
| closable | 可關閉 | bool | false |
| on_close | 關閉回調 | Option\<EventHandler\> | None |
