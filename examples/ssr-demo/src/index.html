About This Demo

This is a complete server-side rendering demonstration using the Hikari UI library. It showcases how to build production-ready web applications with Rust, combining the power of Dioxus for frontend and Axum for backend.

Features

Server-Side Rendering

Full SSR support with Dioxus, enabling fast initial page loads and SEO optimization.

Static Assets

Efficient static file serving with caching headers for JS, CSS, images, and more.

Hot Reloading

Development-friendly hot module replacement for rapid iteration.

Health Checks

Built-in health check endpoints for monitoring and load balancer integration.

CORS Support

Cross-Origin Resource Sharing configured for API access.

Compression

Automatic response compression for reduced bandwidth usage.

Available API Endpoints

Tech Stack

Built with modern Rust technologies:

  • Dioxus 0.7 - Reactive UI framework
  • Axum 0.8 - Web application framework
  • Tower - Modular service abstractions
  • Tokio - Async runtime
  • Hikari Components - UI component library
  • Tracing - Structured logging

Getting Started

Run the server:

cargo run --bin ssr-demo

The server will start on http://localhost:3000
